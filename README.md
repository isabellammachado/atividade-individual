# React + Vite

HeroNet — Mini Rede Social de Super-Heróis

Um projeto simples desenvolvido em React, simulando uma rede social de super-heróis.
Conta com uma página de login, uma página de feed (Home) inspirada no Instagram e uma interface estilizada com Bootstrap e CSS personalizado.

 Funcionalidades:

✅ Página de Login — simples e funcional, apenas para simular acesso.
✅ Página Home — exibe cards com fotos e descrições dos super-heróis.
✅ Navbar — criada com Bootstrap, fixa no topo e com estilo limpo.
✅ Menu Lateral — posicionado com CSS, aparece ao lado da home (como um menu de redes sociais).
✅ Cards de Fotos — componentes independentes com título, imagem e descrição.

Design

Interface leve e inspirada no Instagram
Ícones e imagens personalizadas dos heróis
Layout responsivo e adaptável
Menu lateral feito com position e flex no CSS, posicionado manualmente “na extrema esquerda”
